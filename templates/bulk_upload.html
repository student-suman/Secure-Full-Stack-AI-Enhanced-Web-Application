Bulk Certificate Upload

How to Use Bulk Upload

This tool allows you to upload many certificates at once. Please follow these steps carefully:

  1. Prepare Your Files: Create a folder on your computer containing all the certificate files (PDFs, images, etc.).
  2. Prepare Your Metadata: Download our CSV template and fill it out. Each row represents one certificate. The `filename` column must exactly match the name of the corresponding file in your folder.
  3. Create a ZIP File: Compress the folder containing all your certificate files into a single `.zip` file.
  4. Upload Both Files: Use the form below to upload your completed CSV file and the new ZIP file.
